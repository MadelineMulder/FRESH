About this Project

Natural disasters are formidable events that have caused a lot of havoc around the world. Presently, these disasters have become more persistent and peculiar thus the need to leverage technology to study and analyze the patterns. The aim of this project is to create a python-based web application focused on identifying accessible routes during natural disasters. Specifically, the web application will identify the best path between points during flood events that have been caused by hurricanes. To scale this project appropriately, the web application will focus on one specific area and hurricane, Hurricane Dorian which hit the Grand Bahama Island in 2019.

The app will serve two purposes: analysis and visualization (of routing during a flood event). To fulfill its purpose, the app will include visualizations of various data layers including landcover, flood masks, buildings, and streets as well as a visualization of the hurricane path. Additionally, the application will analyze the best path for a user to take in order to avoid flooded areas and reach their destination as fast as possible. Ideally, users will be able to identify a start point and an end point on the map and the application will analyze flood and network data consequently returning the fastest and most accessible route for users to take between their points.

To accomplish this, group members will use OSMnx for routing analysis as well as Django and Folium to build and visualize a web map of the data layers and routing results. The expected outcome is a web map that includes various data layers, a visualization of Hurricane Dorianâ€™s path, and network analysis capabilities which take into account flood data.

Objectives

  1. To simulate the tracking of the movement of Hurricane Dorian on its path towards Grand Bahama Island
  2. To create an interactive web mapping application that conducts a network analysis of available routes during the flood event caused by Hurricane Dorian
  3. To design a user interactive interface that allows users to choose two points of interest between which the best route will be calculated in the face of the flooding caused by Hurricane Dorian

Case Study
